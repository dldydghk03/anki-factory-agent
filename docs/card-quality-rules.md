# Card Quality Rules

These rules define what a good generated card should look like. They are meant
to be reusable across subjects and lecture styles, not tied to one endocrine
deck or one instructor.

## 1. Choose the Right Recall Target

A cloze should hide the concept that changes the answer.

Prefer hiding:

- diagnosis-defining findings
- differentiating features
- causal mechanisms
- treatment-changing thresholds
- high-yield complications
- test-selection logic

Avoid hiding:

- a number when the concept is more important
- a generic phrase like "common" or "important"
- a word that can be guessed from grammar alone
- a detail that has not been introduced yet

## 2. Define Terms Before Using Them

If a card uses a term such as bilateral, metastatic, localization, incidentaloma,
or catecholamine, the learner should have already seen a card that defines the
term or places it in context.

The agent should insert a short foundation card before frequency tables,
old exam summaries, or advanced comparison cards.

## 3. Keep Cards Connected

Good cards should read like part of a learning sequence.

Weak pattern:

```text
Chromaffin cells contain granules. Chromaffin tumors are brown. Catecholamines
increase blood pressure. Each cell has many granules.
```

Better pattern:

```text
Chromaffin cells store catecholamines in dense granules. These granules react
with chromium salts, which explains the brown color and the name
pheochromocytoma. The stored catecholamines explain the hypertension,
palpitations, and sweating seen in catecholamine-secreting tumors.
```

## 4. Use Natural Language

Generated cards should not preserve awkward source phrasing.

Avoid:

- "is attached"
- "is commonly said separately"
- "when meeting an incidentaloma"
- "first exclude" when the intended meaning is "must be checked first"

Prefer:

- "is characteristic of"
- "is more common in"
- "when an adrenal incidentaloma is found"
- "it is important to check whether this is PPGL first, because..."

## 5. Avoid Repeated Concept Cards

Repeated cards are allowed only when each card has a different job.

Example:

- foundation card: what PPGL means
- diagnosis card: which biochemical test is used first
- risk card: why missing PPGL before surgery is dangerous

Do not create three cards that all say "adrenal incidentaloma requires PPGL
evaluation" unless each one adds a distinct decision point.

## 6. Image Cards Need a Cue and Spacing

Image-based cards should include:

- a short cue explaining what the image is being used for
- spacing between the image caption and the next text block
- no image placed above or overlapping the title
- no dense paragraph immediately attached to the image

The image should support the question, not interrupt reading.

## 7. Old Exam References Should Be Clarified

Exam-reference cards should preserve the useful clue but explain why the clue
matters.

Avoid compressing old references so much that the card becomes a list of
unexplained terms. Add a one-line cue when a visual finding, CT clue, or old
exam phrase is only a supporting clue.

## 8. Promote Rules Only When They Generalize

A rule can be promoted when it improves multiple decks or lecture profiles.

Before promotion, ask:

- Does this help outside the current lecture?
- Does it conflict with image-heavy, case-based, or calculation-heavy decks?
- Can it be expressed as a profile-specific option instead of a global rule?
- Does it reduce false positives or false negatives in review?
